# ЁЯЪА TCS NQT Practice Solutions

Welcome to the **TCS NQT Practice** repository! This project contains a comprehensive collection of Java programs and algorithmic solutions designed to help crack the **TCS National Qualifier Test (NQT)** and other software engineering technical interviews.

## ЁЯОп Overview

This repository is structured to provide logical progression through various problem-solving concepts, ranging from basic mathematics to complex array manipulations and string operations. 

## ЁЯУБ Repository Structure

The project is divided into distinct directories to keep the code logically organized:

### ЁЯМЯ `src/` - Core Practice Problems
Contains a mix of standard interview problems focusing on strings, arrays, mathematics, and logic building.
- **Math/Numbers:** `Fibonacci.java`, `PerfectNumber.java`, `StrongNumber.java`
- **Strings:** `StringPractice.java`, `StringReplace.java`, `LetterFrequency.java`
- **Arrays & Algorithms:** `KadaneAlgo.java`, `MoveZeros.java`, `FindSecondLarge.java`, `PrefixSum.java`
- **Dynamic Programming/Specific Logic:** `ClimbingStairs.java`, `TrainTravelTime.java`, `MountainBikers.java`

### ЁЯдЦ `GPT/` - Structured Learning Paths
This module contains structured, day-by-day practice sessions for systematic learning.

* **`GPT/src/` - Number Theory & Basics**
  Focuses on foundational math algorithms: `Armstrong.java`, `FibonacciSeries.java`, `GCD.java`, `LeastCommonMultipler.java`, `Palindrome.java`, `PrimeORNot.java`, `CountDigits.java`, `SumOfDigits.java`.
* **`Day 2/` - Fundamentals** 
  Contains `Average.java` and `TDSPractice.java`.
* **`Day 3/` - Array Mastery**
  Extensive focus on array manipulation techniques:
  - `KadaneAlgorithm.java` (Maximum Subarray Sum)
  - `RotateArray.java`, `MissingNoArray.java`
  - `LeadersArray.java`, `RemoveDuplicates.java`
  - `PushZeros.java`, `SecondLargeElement.java`, `SecondSmallest.java`
* **`Day 4/` - Advanced Techniques**
  Contains optimization concepts like `PrefixSum.java`.
